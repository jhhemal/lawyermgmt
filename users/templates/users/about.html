The project Lawyer Hiring system includes registration of Lawyer and client’s details. according the curt name, expert area, membership and it store their details into the system. The software has the facility to give a unique id for every lawyer and client and stores the details. It includes a search facility to hire a lawyer at home. User can search availability of a lawyer using curt name, expert area, category, etc. then a client’s get quick consultation from his/her suitable lawyer. The Lawyer hiring System can be entered using a username and password. It is accessible either by an administrator. Only the administrator can make change data into the database. The data can be retrieved easily. The interface is very user-friendly. The data are well protected for personal use and makes the data processing very fast. Lawyer Hiring System is powerful, flexible, and easy to use and is designed and developed to deliver real conceivable benefits to lawyer and clients.
